OMA LwM2M Object Snapshots
==========================

This directory contains GEISA-referenced OMA LwM2M object definitions and
GEISA-proposed object drafts used by this version of the specification.

These files are included to provide a stable specification-version snapshot
for GEISA release review and implementation. They are not intended to replace
the OMA LwM2M registry as the authoritative registry source.

For registered OMA objects, the OMA registry remains authoritative. For
GEISA-proposed objects that are not yet registered, the files in this
directory may represent draft definitions under review.

Schema references
-----------------

The local full-object validation schema in this directory is
``LWM2M-v1_1.xsd``. It preserves the full LwM2M object XML structure used by
the object files in this directory and includes current resource
``Type`` values such as ``Unsigned Integer`` and ``Corelnk``.
